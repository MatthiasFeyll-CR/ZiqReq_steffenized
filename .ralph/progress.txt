- Status: PASS

- Tests: 149 passed (19 test files), including 7 new tests (T-3.2.02, T-3.2.03, visual feedback, nested groups)
- Learnings for future iterations:

  - `findParentGroup` selects the smallest-area group containing the absolute position, which naturally handles nested groups.
  - React Flow handles child-moves-with-parent natively when `parentId` is set on a node.
  - When reparenting, recalculate position relative to new parent (absX - parentAbsX).
  - Set `expandParent: true` on child nodes so the parent group auto-expands if the child is near the edge.
  - Visual drop zone feedback uses `_dropTarget` flag in group node data, toggled during `onNodeDrag`.
  - Docker test container copies files at build time ŌĆö must `docker compose -f docker-compose.test.yml down && up -d --build` after changing test files.
  - Mock `useNodesState` must be stateful (using a module-level variable + callback-aware mockSetNodes) for tests that verify state mutations via setNodes callbacks.
