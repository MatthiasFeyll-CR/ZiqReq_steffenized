  - IdeaKeywords.keywords uses ArrayField (not JSONField)
  - pgvector vector columns require `pgvector` Python package and `CREATE EXTENSION vector` in migration

  - ruff --fix can auto-fix F401 (unused imports) in scaffold files
  - Frontend tsconfig needed `"types": ["vite/client"]` for import.meta.env
  - SQL strings in migrations must use string concatenation to stay under 120 char line limit
